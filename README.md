# Ignites - Tapdown 2D Game

A fun 2D tapdown game where you tap falling objects to ignite them and score points!

## How to Play

1. Open `index.html` in your web browser
2. Click "Start Game" to begin
3. Tap or click on the falling colorful objects to ignite them
4. Each successful tap earns you 10 points
5. The game gets progressively faster as your score increases
6. Try to get the highest score possible!

## Controls

- **Mouse Click**: Tap objects on desktop
- **Touch**: Tap objects on mobile devices
- **Start Game**: Begin a new game
- **Pause**: Pause/resume the current game
- **Reset**: Reset the game and score

## Features

- Colorful falling objects with glow effects
- Particle explosion effects when objects are ignited
- Progressive difficulty (objects fall faster as you score)
- Responsive design for both desktop and mobile
- Score tracking

## Technical Details

- Pure HTML5, CSS3, and JavaScript
- Uses HTML5 Canvas for 2D graphics
- Smooth 60fps animations
- Touch-friendly for mobile devices
